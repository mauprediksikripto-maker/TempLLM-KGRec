# TempLLM-KGRec
TempLLM-KGRec: Temporal Knowledge Graph Reasoning with LLM Integration for Cold-Start Recommendation. PyTorch implementation IJAIN 2026. Jointly optimizes temporal KG snapshots (GRU), LLM-KG alignment (LLaMA-3-8B), dynamic graph propagation (InfoNCE SSL), and schema-conditioned MAML cold-start adaptation.
